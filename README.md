## 🚀 Experiencia y Foco Técnico

### 👥 Liderazgo y Gestión de Equipos
Liderazgo y mentoring de equipos de desarrollo enfocados en la entrega simultánea de múltiples productos críticos (.NET, Angular, Flutter, SQL Server). Especialista en establecer estándares técnicos, implementar procesos de code review y alinear los objetivos de ingeniería con la visión de negocio.

### ⚡ Modernización de Plataformas
Especialista en la migración y evolución de sistemas legacy hacia arquitecturas modernas (ej. .NET Core a .NET 9, Angular v17+, Flutter v3.x), logrando una reducción drástica de la deuda técnica y optimizando la performance en entornos de producción.

### ☁️ Cloud y Arquitectura de Software
Diseño de soluciones escalables utilizando servicios en la nube (Azure App Services, Storage, Azure SQL) y sistemas de mensajería (RabbitMQ, Azure Service Bus). Aplicación rigurosa de patrones de diseño avanzados: CQRS, Clean Architecture, MediatR y documentación mediante ADRs.

### 📱 Ecosistema Mobile
Desarrollo y arquitectura mobile de extremo a extremo con Flutter. Implementación de monitoreo de estabilidad y analíticas con Firebase (Firestore, Crashlytics), gestión de despliegues y distribución automatizada en tiendas de aplicaciones.

### ⚙️ Cultura DevOps
Diseño e implementación de pipelines de CI/CD utilizando GitLab y GitHub Actions, orientados a la reducción de tiempos de deploy y al incremento de la frecuencia y confiabilidad de los releases.
